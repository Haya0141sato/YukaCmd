# YukaCmd

## Overview
bash及びXWindowSystemが使える環境で、wine上などで動かしているVOICEROID+ 結月ゆかりをコマンドラインからしゃべらせるスクリプトです。
MITライセンスで配るので、改修できる方にいろいろなおしてもらえれば嬉しいです。 

##Requirement
xdotool
xsel
xwininfo

##Usage
./YukaCmd.sh [STRING]
./YukaCmd.sh [-f] [FILE]
###example
$./YukaCmd.sh "はじめてのりなっくす"

##Install
Requirementに書いてあるコマンドをインストールし、スクリプトをダウンロードすればいいです。
